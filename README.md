# Resume Builder Web Application

A modern web application that allows users to create professional resumes quickly and efficiently.

## 🚀 Features

* Add personal details, education, and skills
* Dynamic resume preview
* Clean and responsive UI
* Easy-to-use interface

## 🛠️ Tech Stack

* React (Frontend)
* TypeScript
* Tailwind CSS
* Supabase (Backend)

## 💡 Description

This project was developed as part of my learning in web development. It focuses on building a user-friendly interface for generating resumes with real-time data handling.

## ⚙️ How to Run

```bash
npm install
npm run dev
```
